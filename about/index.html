About

This website depends on weather stations that are maintained by third parties. The current main weather station is called the Sonbesie, which is owned by the Mechanical and Mechatronic Engineering department of the Stellenbosch University. Details on this and other weather stations can be found below.

  • Sonbesie - Sensor and Measurement description
  • M&M and E&E - Measurement description
  • Stellenbosch University weather station setup (legacy)
  • Location of weather stations and webcam on Google Maps (outdated)

Interesting facts

  • The first iteration of this website was created in 2008 when the author was a first-year student in Electronic Engineering at the Stellenbosch University.
  • The website is hosted by the Stellenbosch University on a server running the LAMP stack. The current components are:
    • Red Hat Enterprise Linux Server release 7.9 (Maipo)
    • Apache 2.4.34
    • MariaDB 5.5.68
    • PHP 5.4.16
    There is a project underway to migrate to a newer server with:
    • Red Hat Enterprise Linux release 8.10 (Ootpa)
    • Apache 2.4.37
    • MariaDB 10.3.39
    • PHP 7.3.20
  • The software behind this website is open source and available on Github . Please feel free to contribute.

Contact

This website is operated on a best-effort basis. Its uptime is not constantly monitored. In case of an outage, please check back later. If the issue persists, please contact us below. We are not able to provide support regarding the data obtained from this site.
